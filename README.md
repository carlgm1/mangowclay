# mangowclay
A layout changing script for mangowc - 
If run with no parameters, will cycle to next layout. If "select" is passed in as parameter, will display a list in rofi. At the end of the script it will write tag rules to a file (~/.config/mango/tagrules.conf) that can be sourced in whatever is being used as mango config file. This will preserve the layouts on each tag between logins (maybe there is a way that this can already be done, or will be a way, which would make that part redundant).
